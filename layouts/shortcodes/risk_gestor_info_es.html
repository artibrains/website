Simulador: El Gestor de Riesgos

Tu misión es encontrar el umbral de decisión perfecto para un modelo de IA que predice si un paciente faltará a su cita. El objetivo es minimizar el coste total para la clínica.

¿Cómo funciona esto?

El Modelo de IA (simulado)

Imagina que un modelo de Machine Learning ha analizado miles de datos históricos. Para cada uno de los 100 pacientes de abajo, ha calculado la probabilidad de que falte a su cita. El modelo es bueno, pero no perfecto: los pacientes que realmente faltan tienden a tener una probabilidad más alta.

Tu Papel como Gestor

El modelo no toma la decisión final. ¡Ese eres tú! Usando el slider, defines el umbral de riesgo. Si la probabilidad de un paciente supera tu umbral, el sistema lo clasifica como "probable ausente" y le envía un recordatorio (coste: 5€).

El Objetivo del Juego

Encuentra el punto dulce. Un umbral muy bajo significa muchos recordatorios innecesarios. Un umbral muy alto significa perder pacientes que no fueron avisados (coste: 25€ por cita perdida). ¡Observa el "Coste Total" y busca el valor mínimo!
